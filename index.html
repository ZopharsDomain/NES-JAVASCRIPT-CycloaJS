JavaScript NES Emulator: CycloaJS

Please drag & drop your .NES game or select opensource game below.

.NESファイルをドラッグ・アンド・ドロップするか、以下からフリーなゲームを選択すると遊ぶことができます。

This emulator is a port of Cycloa, a NES emulator written in C++, to JavaScript. The main purpose of this development is survey JavaScript performance.

このエミュレータは、C++で書かれたCycloaというエミュレータのJavaScriptへの移植です。開発の主目的はJavaScriptのパフォーマンスの調査です。PC上においては概ね実用的なスピードが出せるものの、、サウンドの同期機構やFPSの制御が難しく、
